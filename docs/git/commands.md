title: Regular Commands
---

```sh
git add <file_path>
git add -A
git commit -am 'commit msg'
git push origin <branch_name>
git rebase main
git checkout -b <New_branch_name>
git branch -D <branch_name_to_delete>
git stash
git stash pop
git reset --hard HEAD~1 # to revert the latest commit
```
